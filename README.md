## 编译方法
- 安装依赖
```
yarn install
```

- 开发模式运行
```
yarn run serve
```

- 编译项目
```
yarn build
```
- node版本

v14.17.3

- pdf-dist版本

"2.16.105"
支持在线批注pdf、下载等
