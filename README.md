## 编译方法
- 安装依赖
```
yarn install
```

- 开发模式运行
```
yarn run serve
```

- 编译项目
```
yarn build
```
- node版本

v14.17.3

- pdf-dist版本

"2.16.105"

- 简介
支持在线画线、画圆、画矩形、添加文字到pdf并生成可用于保存的json数据、生成新的pdf下载等 1.0版本初步完成(暂无json数据回显功能，后期添加)
